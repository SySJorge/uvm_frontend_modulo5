# UVM Frontend Módulo 5

Repositorio de actividades del módulo 5: Build Tools, Webpack y Service Workers.

## Estructura

- Tema2: Webpack básico y generación de bundle.
- Tema3: Integración de CSS, activos y datos con Webpack.
- Tema4: React y Babel con Webpack.
- Tema5: TypeScript y SASS con Webpack.
- Tema6: Service Workers con Workbox.

Cada carpeta contiene su propio proyecto con archivos de código, configuración de Webpack y package.json.
